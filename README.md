
# # simple Image Gallery

Using React & MomentJs.
You can check **[DEMO](https://shayanypn.github.io/simple-gallery/)**


> The demo run on GH Pages


Quick Start
-----------


```shell
$ git clone https://github.com/shayanypn/simple-gallery.git
$ cd simple-gallery
$ npm install
$ npm run start
```
